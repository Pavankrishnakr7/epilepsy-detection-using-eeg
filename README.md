# 🧠 Epilepsy Detection Using EEG Signals

## 📌 Overview
This project focuses on detecting epileptic seizures using EEG (Electroencephalogram) signals with the help of deep learning techniques. A CNN-LSTM hybrid model is used for accurate classification.

---

## 🎯 Objective
To build an AI-based system that can automatically detect epilepsy from EEG signals and assist in early diagnosis.

---

## 🛠️ Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib

---

## 🧪 Methodology
1. Collected EEG dataset
2. Preprocessed signals (noise removal, normalization)
3. Extracted features
4. Built CNN-LSTM model
5. Trained and evaluated model

---

## 📊 Results
- Achieved 93% accuracy** (update this)
- Efficient classification of seizure vs non-seizure signals

---

## 🚀 Features
- Automated seizure detection
- Deep learning-based classification
- Signal preprocessing pipeline
